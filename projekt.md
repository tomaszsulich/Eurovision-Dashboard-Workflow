# Eurovision Dashboard Project



## Project description



The goal of this project is to create an interactive Eurovision dashboard presenting voting results, country statistics and contest trends across different years and decades.



The dashboard will focus on:



* jury and televoting analysis,

* country performance comparisons,

* voting patterns between countries,

* historical Eurovision trends,

* interactive data visualizations.



The project is intended as both a data analysis practice project and an opportunity to improve Git and GitHub workflow skills.



## Planned technologies



* Power BI

* Git \& GitHub

* CSV/XLSX datasets



## Planned features



* Interactive filters by year, decade and country

* Heatmaps and score comparisons

* Jury vs televoting analysis

* Historical statistics dashboard

* Exportable reports and screenshots
